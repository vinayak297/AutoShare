AutoShare is a web app that helps commuters quickly find co-passengers for shared auto rides. A user at an auto stand can broadcast a ride request, allowing nearby commuters to join the ride. The app shows each participant’s distance from the stand and automatically closes the request once the auto capacity is filled. This reduces waiting time, lowers travel costs, and improves urban ride-sharing efficiency.
